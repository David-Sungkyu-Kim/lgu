데이터 자동 주기 안내

1. 데이터 자동 주기 안내
데이터 자동주기는 가족결합 회선에 한해 최대 4명에게
전달할 수 있습니다. (1924MB)
또한 가족간 데이터 주고받기 횟수(최대 4회)에
포함됩니다.
2. 데이터 선물 내역 확인
매월 1일 자동으로 나눠주기
[우측 상단 메뉴 버튼 선택 > 이용내역] 페이지
에서 확인하실 수 있습니다.
3. 데이터 자동주기 취소
[우측 상단 메뉴 버튼 선택 > 이용내역] 페이지 의
“데이터 자동주기 내역”의 “설정해제” 버튼 을 선택하시면
익월 1일 부터 데이터 자동주기
예약이 해제됩니다.
상세안내 > 단, 전달된 데이터는 취소 불가합니다
공지사항 바로가기
