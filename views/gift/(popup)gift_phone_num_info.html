데이터 자동주기 안내

고객님의 데이터 1,000MB가
홍*동 님(010-1234-1234)에게
매월 1일 마다 예약 전달됩니다.

  • 1데이터 자동 주기 안내

    데이터 자동주기는 가족결합 회선에 한해
    최대 2명에게 전달할 수 있습니다.
    또한 가족간 데이터 주고받기 횟수
    (최대 4회)에 포함됩니다.

  • 2데이터 선물 내역 확인

    [우측 상단 메뉴 버튼 선택 > 이용내역]
    페이지에서 확인하실 수 있습니다.

  • 3데이터 자동주기 취소

    데이터 자동주기를 이미 설정한 경우,
    [우측 상단 메뉴 버튼 선택 > 이용내역]
    페이지에서 데이터 자동주기를
    취소하실 수 있습니다.
    익월 1일부터 데이터 자동주기 예약이
    해제되나 이미 전달된 데이터는
    취소 불가합니다.
